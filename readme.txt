== Description ==

moreAds SE is a plugin to deliver ads on your website.
Get the maximum out of your traffic by customizing your ads through a precise user-targeting for your target group.
Ads will also be displayed, even when visitors use an ad blocker.

Ad zones are implemented as WordPress widgets.

= Supported ads =
*   Banner Ads (PNG, JPEG, GIF)
*   HTML Ad (Custom HTML code)
*   Popup Ad (Url only)

= Supported ad zones/widgets =
*   Banner Zones
*   Floating Banner Zones
*   Popup Zones
*   TextLink Zones

= Available in the following languages =
* English
* German

= Successfully tested caching plugins =
* WP Super Cache

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. That's it
